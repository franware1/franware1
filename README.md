# What I'm working on right now!

I am currently working on:

- An AI aide for electric grid operators to optimize and ensure stability in the electric grid.
With increasing power demands from the growing AI industry using hungry data centers, the electric grid is going to 
undertake a lot more load in these coming years. What better idea is there than to allieviate this issue with AI itself!

- A developer portfolio website that showcases my network, skills, and projects. I'm using React to build the components and
Typescript to catch the bugs of the website and plain CSS to style the website (I did import Tailwindcss but I've decided to just use plain CSS).
